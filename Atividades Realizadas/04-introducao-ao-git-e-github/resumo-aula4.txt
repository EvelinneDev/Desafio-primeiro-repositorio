Git e Github

O Git é um sistema de versionamento de código concluído.
Criador: Linus Torvalds(criou devido ao desgosto de usar outros sistemas de versionamento existentes. ex:Kernel)

Github

Repositório online de códigos. Funciona também como uma espécie de rede social.
vantagens em utilizar:
1- Controle de versão
2- Armazenamento em nuvem
3- Trabalho em Equipe
4- Melhoras seu código
5- Reconhecimento
