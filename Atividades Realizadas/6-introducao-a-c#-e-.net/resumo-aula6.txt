Introdução ao C# e .NET

.NET: é um framework criado pela microsoft que permite a comunicação entre várias linguagens
que tem uma infraestrutura comum(CLI), entre elas o Visual Basic, C++, F# e C#.

Framework: É um conjuto de cógigos prontos que facilitam o desenvolvimento disponibilizando algumas bibliotecas.

C#: Linguagem orientada a objetos fortemente tipada.
A sintaxe orientada a objetos foi baseada no C++ e possui influências de outras linguagens de programação como Object, Pascal e Java.


Criando o primeiro projeto

Pra começar, é preciso criar um porjeto e escolher um template que já vai ser criado com algumas pré configurações.
Então, vamos definir o nome do projeto e o nome da solução, que por padrão vão ser os mesmos.

Solução: É um container para organizar os projetos relacionados.

Os diretórios e arquivos do projeto podem ser vistos pelo gerenciador de soluções,
que podem ser acessados pelo menu exibir.

No gerenciador de soluções, podemos adicionar mais projetos, classes ao projeto, diretórios e mexer no projeto de forma geral.

Estruturas

Namespace: São usados para organizar classes.
Para usar uma classe de outro namespace, colocamos using no começo do código.
Não é indicado utilizar duas classes no mesmo Namespace

Classe

Uma classe pode ter campos, propriedades, métodos e eventos dentro dela que são denominados membros.

Interface
É como uma base que serve de molde para várias classes.
Uma classe que implementa uma interface deve implementar todos os membros.

Enum
Declara um conjunto de constanstes nomeadas que começam do 0 e aumentam de 1 em 1.

Debugging

A depuraçõa pode ser feita colocando um Breakpoint no canto esquerdo do código ou apertando f9 e executando ele.
O código vai parar onde foi colocado o BreakPoint e então o código poderá ser inspecionado.
